# 🧪 Software Testing using Selenium WebDriver

This project demonstrates **automated web application testing** using **Python** and **Selenium WebDriver**. The objective is to replace repetitive manual testing with automated test scripts, improving testing speed, accuracy, and reliability.

The project includes automated test cases for **Swag Labs** and **CandyMapper**, covering common user workflows and UI functionality.

---

## 📖 Project Overview

Software testing plays an important role in ensuring that web applications function correctly before deployment. Instead of manually repeating the same tests, Selenium WebDriver automates browser interactions by simulating real user actions such as clicking buttons, entering text, selecting options, and verifying expected outcomes.

This project focuses on **Functional Testing** and **UI Testing**, ensuring that different features of the websites behave as expected.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/Software-Testing.git
cd Software-Testing
```

Install dependencies

```bash
pip install selenium webdriver-manager
```

---

## ▶️ Run the Tests

Execute the Python test script:

```bash
python test_cases.py
```

or run directly from **PyCharm**.
